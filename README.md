## 🎯 About

> Hands-on Deep Learning experiments covering **ANN, CNN, Transformers, and real-world image classification projects** using Keras, TensorFlow, and PyTorch.

---

## 📂 Repository Structure

```
deep-learning-lab/
│
├── 📁 notebooks/                              # Core DL Notebooks
│   ├── Artificial_Neural_Networks.ipynb        # ANN fundamentals
│   ├── Backpropagation-py.ipynb                # Backprop from scratch
│   ├── Classification-with-Keras.ipynb         # Multi-class classification
│   ├── Convolutional-Neural-Networks.ipynb     # CNN architecture
│   ├── Regression-with-Keras.ipynb             # DL regression
│   └── Transformers-with-Keras.ipynb           # Transformer architecture
│
├── 📁 projects/
│   ├── chicken-disease-classification/         # 🐔 Production CNN Pipeline
│   └── aircraft-damage-classification/         # ✈️ Damage Detection
│
└── 📁 docs/
    └── learning_notes.md
```
